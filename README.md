# Simple Load Balanced ACI Containers based on HAProxy

## Use
- Fill in variables in vars.env
- `./scripts/storage.sh`
- `./scripts/mount.sh`
- `cd Dockerfile`
- `../scripts/build.sh`
- `../scripts/deploy.sh`
